conga-twig
==========

Overview
--------

This is a bundle for the [Conga.js](https://github.com/congajs/conga) framework which 
integrates the [twig.js Template Engine](https://github.com/justjohn/twig.js) into a project.

Configuration
-------------

Example:

    // config.yml
    twig:

        options:
        	strict_variables: false
